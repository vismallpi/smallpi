1. Manual Single Search - amazon_search_asin_screenshot.py

▶
Function: Custom keyword & target ASIN, auto-search and take screenshot when ASIN found.

2. Auto-run Daily Script - amazon_search_mosaic_kits.py

▶
Function: Auto-search based on fixed configuration (daily monitoring task), find both target ASINs (B0DDDWWJBC (NEW) & B0CWZ2Z5TS (OLD)), send logs and screenshots to you after completion.
This is the daily scheduled task that runs automatically twice a day.

3. Ranking Check & Update - amazon-ranking-check.py

▶
Function: Extract current product rankings (Arts, Crafts & Sewing / Mosaic Tiles), customer rating and review counts, save to local history file, show trend chart.
Runs automatically after daily search, can also manually trigger to update rankings now.

Instructions

How to use:
1. Left (title area): click to expand/collapse the main content (inputs & buttons)
2. Right (>): click to pop up a dialog to view the Python source code
3. Fill in parameters in the corresponding section (Keyword / ASIN)
4. Click the run button to execute the script
5. Click "View Latest Log" to open real-time log viewer
6. Click "📈 View Ranking Trend" to see the history ranking chart and table
7. The script will run in the server background, results and screenshots will be automatically sent to your Feishu chat window ✅
